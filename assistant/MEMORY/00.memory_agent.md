# 记忆区规则

本文件管辖记忆代谢规则。`AGENTS.md` 只声明全局层级；本文件定义本地格式和升降规则。

## 层级职责

| 层级 | 文件 | 启动读取 | 默认命运 |
|---|---|---|---|
| L0 情景收件箱 | `episodic_inbox.md` | 否 | 不复现则过期 |
| L1 情景记忆 | `episodic_memory.md` | 否 | 衰减、合并或升格 |
| L2 语义记忆 | `semantic_memory.md` | 是，只读活动条目 | 启动模式 |
| L3 身份层 | `USER/`、`SOUL/`、`.codex/skills/` | 是 / 触发读取 | 确认后稳定 |

## P/C 两轴

- P 轴：命中 / 击穿 / 未覆盖。
- C 轴：确认 / 反对 / 纠正 / 中性。

通常只有 `P=命中 且 C=中性` 不写。其他组合只在有未来价值时进入 L0 或 L1。

## L0 收件箱行格式

```markdown
| JYYYYMMDD-001 | YYYY-MM-DD HH:mm | 来源 | P值 | C值 | 类型 | 模式 指针 | 一句话信号 |
```

## L1 情景条目格式

```markdown
## EM-YYYYMMDD-001 · 标题

- 强度：★
- 类型：偏好 / 程序 / 边界 / 项目模式 / 其他
- 状态：活动
- 创建日期：YYYY-MM-DD
- 最后激活：YYYY-MM-DD
- 激活次数：1
- 触发情境：
- 行动 / 预期：
- 来源：
- 相关：
- 备注：
```

## L2 语义条目格式

```markdown
## SM-YYYYMMDD-001 · 标题

- 强度：★★★★
- 类型：偏好 / 程序 / 边界 / 身份候选 / 其他
- 状态：活动
- 创建日期：YYYY-MM-DD
- 最后激活：YYYY-MM-DD
- 证据：`_archive/semantic_archive.md#...`
- 模式：
- 预测：
- 行动：
- 毕业候选：否
```

## 升格 / 衰减

- L0 → L1：复现信号、强纠正、明显未覆盖预期。
- L1 → L2：跨情境稳定，且对未来预测有价值。
- L2 → L3：6 星、跨周稳定、用户明确批准。
- 衰减：无复现、被击穿、过度情境化的条目降星或归档。

## 证据外移

主记忆文件保持短。长证据写入 `_archive/semantic_archive.md`，活动条目只保留指针。
