# 02 阅读区 agent

The reading area stores reading notes that are not yet part of a specific project.

## Rules

- Keep source metadata visible.
- Move project-relevant notes into the project `_reference/` area when they become evidence.
- Do not let reading notes become the only location for project conclusions.
