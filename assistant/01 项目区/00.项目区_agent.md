# 01 项目区 agent

The project area owns sustained work. Each project should have an `_overview.md` and, when useful, a `_progress/` folder.

## Project Overview Minimum

- project definition
- current state
- loading chain
- progress index
- open questions

## Rules

- Main documents hold stable conclusions.
- `_progress/` holds reasoning process and node-level records.
- `_reference/` holds source records when research is involved.
- Cross-project implications should use pointers rather than duplicated content.
