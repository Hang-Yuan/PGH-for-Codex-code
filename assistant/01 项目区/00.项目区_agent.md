# 01 项目区 agent

项目区管辖持续推进的工作。每个项目至少应有 `_overview.md`；需要记录过程时建立 `_progress/`。

## 项目 overview 最小内容

- 项目定义
- 当前状态
- 加载链
- 推进索引
- 待解问题

## 规则

- 主文档放稳定结论。
- `_progress/` 放推理过程和节点记录。
- `_reference/` 放文献和资料记录。
- 跨项目影响用指针连接，不重复定义。
